Overview

Taskenuis is an intelligent task management system enhanced with artificial intelligence.
It helps users organize, prioritize, and automatically receive task suggestions based on their most productive time periods.
The system also includes fast indexed search, allowing instant access to tasks even in large datasets
Key Features

Create, update, and delete tasks
Due dates and priority management
Task categorization using tags and categories
AI-based task suggestions depending on productive time
Indexed quick search for instant task retrieval
Task status tracking (to-do, in progress, completed)
Smart notifications and reminders
Responsive and user-friendly interface
AI & Intelligent Components
Analysis of user activity to determine productive time slots
Task recommendation based on productivity patterns
Python-based AI processing integrated with the backend
Adaptive suggestion system improving over time

Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
AI / Data Processing: Python
Database: SQL
Version Control: Git, GitHub
